# Google Places API キーの取得手順

写真取得を確実にするため、Places API キーの取得手順です。

## 1. Google Cloud Console にアクセス

https://console.cloud.google.com/

- Google アカウントでログイン

## 2. プロジェクトを作成

1. 画面上部のプロジェクト選択 → **「新しいプロジェクト」**
2. プロジェクト名を入力（例: `gbp-image-app`）
3. **「作成」** をクリック

## 3. Places API を有効化

1. 左メニュー **「APIとサービス」** → **「ライブラリ」**
2. 検索ボックスに **「Places API」** と入力
3. **「Places API (New)」** をクリック
4. **「有効にする」** をクリック

> 注意: 「Places API」ではなく **「Places API (New)」** を有効にしてください。

## 4. API キーを作成

1. 左メニュー **「APIとサービス」** → **「認証情報」**
2. **「認証情報を作成」** → **「API キー」**
3. 作成されたキーが表示されます（`AIza...` で始まる文字列）
4. キーをコピー

## 5. アプリで使用

1. Streamlit アプリを起動
2. サイドバーの **「Google Places API キー」** に貼り付け
3. 「取得開始」をクリック

---

## よくあるエラー

### 「APIキーが無効か、Places API が有効化されていません」
- **Places API (New)** が有効か確認
- 正しい API キーをコピーしているか確認（前後のスペースに注意）

### 「請求先アカウントが必要です」
- Google Cloud は無料枠がありますが、クレジットカード登録が必要な場合があります
- 無料枠内であれば課金されません

### キーを紛失した場合
- Cloud Console → 認証情報 → 該当キーをクリック → キーを再表示
